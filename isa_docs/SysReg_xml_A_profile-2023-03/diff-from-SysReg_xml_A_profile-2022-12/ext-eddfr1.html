EDDFR1, External Debug Feature Register 1

The EDDFR1 characteristics are:

Purpose

Provides top level information about the debug system in AArch64.

Configuration

WhenThe FEAT_DoPDpower isdomain implemented,of EDDFR1 is in the Core power domain. Otherwise, it is IMPLEMENTATION DEFINED whether EDDFR1 is implemented in the Core power domain or in the Debug power domain..

Attributes

EDDFR1 is a 64-bit register.
