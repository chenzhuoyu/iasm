GCSSTR

Guarded Control Stack Store stores a doubleword from a register to memory. The address that is used for the store is calculated from a base register.

Integer
(FEAT_GCS)

313029282726252423222120191817161514131211109876543210
1101100100011111000011RnRt
opc

GCSSTR <Xt>, [<Xn|SP>]

if !HaveGCS() then UNDEFINED; bits(2) effective_el; integer n = UInt(Rn); integer t = UInt(Rt); effective_el = PSTATE.EL;

Assembler Symbols

<Xt>

Is the 64-bit name of the general-purpose register to be transferred, encoded in the "Rt" field.

<Xn|SP>

Is the 64-bit name of the general-purpose base register or stack pointer, encoded in the "Rn" field.

Operation

bits(64) address; bits(64) data; if effective_el == PSTATE.EL then CheckGCSSTRTrap(); AccessDescriptor accdesc = CreateAccDescGCS(effective_el, MemOp_STORE); if n == 31 then CheckSPAlignment(); address = SP[]; else address = X[n, 64]; data = X[t, 64]; Mem[address, 8, accdesc] = data;
